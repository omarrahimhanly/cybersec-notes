# cybersec-notes
Hello guys, it is currently day 3 of learning cybersecurity as a teenager and trying to land a job in this field. 
## What I've Learned So Far

I'm currently documenting my journey into cybersecurity, focusing on building strong networking and Linux fundamentals before moving into more advanced security topics.

So far, I've studied networking through TryHackMe and NetworkChuck's CCNA course, learning how devices communicate across networks, how switching and routing work, the purpose of DNS, and the role of common protocols such as TCP and HTTP. To better understand these concepts, I used Wireshark to capture and analyze real traffic, following the complete process of a web request from DNS resolution to the TCP handshake and HTTP communication.

I've also started working with Kali Linux and various security tools, gaining hands-on experience with wireless networking concepts and learning how wireless adapters interact with operating systems. Through experimentation, I discovered the limitations of virtual machines for wireless monitoring and learned why direct hardware access is often required for tasks involving monitor mode.

Beyond the technical concepts themselves, I've spent time troubleshooting systems, configuring bootable Linux environments, and becoming more comfortable working from the command line. This repository serves as a record of what I learn, the labs I complete, the mistakes I make, and the lessons I take away from them as I continue developing my cybersecurity skills.
